# ⚡ 加入服务器八股文
> 看完了不会还有人不会吧:(

---

在加入服务器之前，您需要满足以下条件：

- 拥有一台个人电脑
- 下载了**服务器专用整合包**
- 拥有 [Littleskin](https://littleskin.cn) 账户且创建了角色
- 您的 Minecraft 账户名不为中文且**字数在 3-16 之间**

接着使用您的 Littleskin 账户验证启动服务器专用整合包

:::warning 警告
因为不使用服务器专用整合包导致的问题，OFS QQ 群聊的 成员/管理员 **不会为您提供任何技术支持**！
:::

---

启动游戏后进入「多人游戏」页面，点击「添加服务器」并在服务器地址一栏输入：```mc-survival.ofs-union.eu.org```，点击完成后双击刚刚添加的服务器即可成功加入服务器
